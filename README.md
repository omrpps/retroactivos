# Retroactivos BBS - Juegos de Guerra

![Retroactivos BBS](images/logo.png)

Una experiencia web de simulación de terminal retro, rindiendo tributo a los Bulletin Board Systems (BBS) de los años 80 y 90, e inspirada por la estética de "Juegos de Guerra" (WarGames).

👉 **[Juega en vivo aquí: Retroactivos en GitHub Pages](https://omrpps.github.io/Retroactivos/)**

## 💻 Características

Esta web abandona la interfaz gráfica moderna para sumergir al usuario en una terminal conversacional donde debe usar comandos de teclado.

- **Fósforo Verde Auténtico**: Interfaz rediseñada por completo utilizando tipografía retro `VT323` mediante Google Fonts.
- **Efectos CRT Reales**: Capas de CSS avanzado añaden líneas de escaneo (*scanlines*), un sutil parpadeo (*glow*), y reflejan la pantalla para imitar la curvatura de los viejos monitores de tubo.
- **Sistema WOPR Inteligente**: En lugar de navegar con botones, la página te exige conversar. Escribe comandos como `ayuda`, `juegos`, `reseñas` o `noticias` para navegar por la base de datos de los proyectos desarrollados por el estudio Retroactivos.
- **Módem de 300 Baudios**: El texto no aparece de golpe, sino que se tipea dinámicamente letra por letra simulando una conexión real por puerto serie.
- **Diseño Inmersivo**: Desde la animación *[NO CARRIER]* al escribir "adios", hasta ventanas de cookies de estilo ANSI ASCII.

## 🕹️ Comandos Disponibles

Para interactuar con la terminal una vez conectado, asegúrate de utilizar el teclado:

* `AYUDA`: Muestra las opciones y secciones del sistema.
* `JUEGOS`: Muestra el catálogo de descargas de Pentio, The Last Magic Forest y Time.
* `RESEÑAS`: Extrae enlaces a los principales medios que cubrieron nuestros lanzamientos (Retromaniac, Indie Retro News, etc.).
* `NOTICIAS`: Lista artículos y reportajes publicados sobre nosotros.
* `PRIVACIDAD`: Despliega nuestro manifiesto y archivo confidencial sobre Cookies y Políticas de Privacidad.
* `ADIOS`: Finaliza tu sesión de protocolo.

## 🛠️ Tecnologías

- **HTML5 & CSS3** (Vanilla)
- **Javascript Moderno** sin frameworks para la lógica de terminal y el motor de dibujo.
- Tipografía `VT323` (Google Fonts).
